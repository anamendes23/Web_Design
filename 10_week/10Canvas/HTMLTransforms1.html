CSS3 Transforms

CSS3 transforms allow you to translate, rotate, scale,and skew elements.
It lets an element change shape, size, and position.

The following are 2D transformation methods:
  • translate()
  • rotate()
  • scale()
  • skewX()
  • matrix()

translate() Method

The translate() method moves an element from its current position.
translate(50px,100px); moves the elements 50 px to the raght and 100px down from its current position


rotate() Method

The rotate() method rotates an element clockwise (positive value), or counter-clockwise(negative value) according to a given degree.
Example: rotate(30deg), will rotate an element clockwise by 30 degree, the center of ratation is the center of the element.

rotating this element by 30 degrees (clockwise)
rotating this element by -30 degrees ( counter clockwise)


scale() Method

The scale() method increases or decreases the size of an elemnt

Example: scale(2,3) increases the element to be 2 times of its original width, and 3 times of its original height

You could also use values less than 1 to scale an element down or make it appear smaller

SkewX() Method

Skews an element along the X-axis by a given angle

skewX(35deg) skews the element 35 degrees along the X-axis

SkewY() Method

Skews an element along the Y-axis by a given angle

skewY(35deg) skews the element 35 degrees along the Y-axis

This div element has been skewed aling the y-axis by 20 degrees

skew() Method

Skews an element along the X-axis Y-axis by a given angle

skew(20deg,15deg) skews the element 20 degrees along the X-axis, and 15 degree along the Y-axis

This div element has been skewed along both the y-axis and the y-axis


Combining transforms

We can combine transformations such as:
transform: rotate(25deg) scale(0.75);
transform: skew(10deg,20deg) translate(30px);

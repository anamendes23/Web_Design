Canvas Transformations

  • scale() : scales the current drawing biggier or smaller
  • rotate(): rotates the current drawing
  • translate(): remaps the (0,0) position on the canvas
  • transform(): replaces the current transformation matrix for the drawing
  • setTransform(): Resets the current transformation to the identity matrix, then runs transform
scale()

It scales the drawing by some factor multiplier like: 2,0.5,...
The position will also be scaled. The scale method affect current and future drawings only.

rotate(angle)

The rotate() methos rotates the drawing clockwise by an angle specified in radians. The rotation occurs with respect to the canvas origin. By default the canvas origin is on the upper left corner of the canvas.
The rotate()method does not affect any drawing already on canvas. it affects only drawings after the rotate method has executed.

Translate(x,y)

The translate() method remaps the (0,0)origin position on the canvas.
It moves or translate the origin to the values set by x and y.
All subsequent drawings will be translated (or shifted) in the x direction set by the x value, and in the y-direction set by the y valuel

HTML Lists

Unordered HTML List

An unordered list starts with the <ul> tag. Each list item in the list starts with the <li> tag.

Example:
  • Coffee
  • Tea
  • Milk
  • Lemonade
  • Water

Ordered HTML List

An ordered list starts with <ol> tag. Each list item in the list starts with the <li> tag.

Example:
  1. Coffee
  2. Tea
  3. Milk
  4. Lemonade
  5. Water

Nested HTML Lists

Example:
  1. Drinks
    1. Coffee
    2. Tea
    3. Milk
    4. Lemonade
    5. Water
  2. Desserts
    1. apple pie
    2. lemon pie
    3. chocolate ice cream

HTML Description List

Description list is accomplished by using 3 tags: <dl>, <dt>, and <dd>

Example:
Coffee
Columbian coffee is the best
Milk
Whole milk is better for you
lemonade
Can be too sweet

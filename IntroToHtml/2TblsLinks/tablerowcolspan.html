Span of Column & Row

Col Span

syntax: <td colspan="number">
where number specifies the number of columns a cell should span.

Example:

colspan example
header1 header2 header3 header4
data 2,1 data 2,3 data 2,4
data 3,1 data 3,2 data 3,3 data 3,4
data 4,1
data 5,1 data 5,2 data 5,3 data 5,4

Row Span

syntax: <td rowspan="number">
where number is the number of rows a cell should span

Example:

rowspan example
header1 header2 header3 header4
data 2,1 data 2,2 data 2,3 data 2,4
data 3,2 data 3,4
data 4,1 data 4,2 data 4,4
data 5,1 data 5,2 data 5,4

Both rowspan & colspan

You can use rowspan and colspan at the same time in any table.
You can apply rowspan and colspan to any <td> tag to expand a cell both row wise and column wis.

Example:Expand cell data 2,2 to take up six cells (2rows by 3cols)

rowspan & colspan example
header1 header2 header3 header4
data 2,1 data 2,2
data 3,1
data 4,1 data 4,2 data 4,3 data 4,4
data 5,1 data 5,2 data 5,3 data 5,4

HTML Structure

Xenia is Bootstrap 3.1 based responsive theme. You can use all of bootstrap features. More information Bootstrap 3.1.

Customization

Xenia has easy customization features.

Color

Open style.css file and change this color #00C0E1 or connect in <head> ready CSS color scheme </head>

Example: '<link href="assets/css/layouts/blue.css" media="screen" rel="stylesheet" type="text/css">'

Background Images

Open style.css, find 15 line (body) and change path to the your BG

Example: body {background: #fff url(../img/YOUR_BG.png) fixed;}

Footer Dark Skin

Connect in <head> ready CSS dark-skin footer </head>

Example: '<link href="assets/css/layouts/dark-skin.css" media="screen" rel="stylesheet" type="text/css">'

Animation

Example for usage.

<div class="animated-area">
   <div class="animated" data-animation="bounceIn" data-animation-delay="0.3s"><img src="1.jpg"/></div>
   <div class="animated" data-animation="bounceIn" data-animation-delay="0.6s"><img src="2.jpg"/></div>
</div>

Classes:

- .animated-area: Wrapper div for animated area.

- .animated: Class for each animate item.

Data attributes:

- data-animation: You can use all of animate.css effects. More information http://daneden.me/animate/

- data-animation-delay: Delay time for your animations.

Flickr Widget

For account change, insert your ID after "user=YOUR ID". If you changed ID for other URL name, generate it on a service http://idgettr.com/

Example: '<script type="text/javascript" src="http://www.flickr.com/badge_code_v2.gne?count=6&display=latest&size=s&layout=x&source=user&user=34328769@N02">'

PHP Mail Form

Open assets/xml/config.php and change the mail address from line 6. Work only in FTP.
